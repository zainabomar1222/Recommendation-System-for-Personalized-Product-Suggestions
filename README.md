# Recommendation-System-for-Personalized-Product-Suggestions
This project was developed for Algorithm Analysis and Design course
## Objective
Compare two recommendation algorithms and determine which one performs better.
## Algorithms
* User-Based Collaborative Filtering (Naive Algorithm)
* Matrix Factorization (Optimized Algorithm)
## Files
* `UserBasedCF_NaiveAlgorithm.py`
* `MatrixFactorization_OptimizedAlgorithm.py`
* `Experiment.ipynb`
## Result
The algorithms are evaluated and the optimized approach is selected based on performance.
